## üí° Usage Tips
1. Run `-listemojis` command to see all available emojis in your Discord servers
2. Copy emoji codes directly from Discord using `\:emoji_name:`
3. Update this file and src/config.js when emojis change
4. Format: `<:name:id>` for static, `<a:name:id>` for animated emojis
5. If emoji doesn't display, check that the bot has access to the server with that emoji
6. Use `-uploademojis` to upload all emojis to your server

## ğŸš€ Quick Setup Steps:

1. Download emoji pack or create emojis (128x128px PNG)
2. Create Discord server called "Bot Emojis"
3. Upload all emojis (Server Settings > Emoji)
4. Name them as listed above
5. Invite your bot to that server
6. In Discord, type \:emoji_name: to get ID
7. Copy IDs and update src/config.js
8. Restart bot: npm start or pm2 restart discord-music-bot
9. Test with /play command to see emojis
